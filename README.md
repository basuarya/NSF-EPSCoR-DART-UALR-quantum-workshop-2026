# Quantum Computing and Quantum Machine Learning Workshop — Timeline & Workshop Patterns

This GitHub Pages-ready package brings the two-day video journey and the earlier public aggregate analytics into one DM Sans interface.

## Contents

- `index.html` — combined timeline, video player, view switcher, and analytics dashboard
- `assets/workshop-banner.png` — workshop banner
- `.nojekyll` — keeps GitHub Pages from applying Jekyll processing

## Views

- **Session timeline** — all 23 Google Drive recordings across Day 1 and Day 2, with an all-sessions view, day filters, search, and focused playback.
- **Workshop patterns** — rounded public indicators for reach, affiliation mix, participation mode, credential pathways, geographic representation, and feedback themes.

Direct links are supported:

```text
YOUR_SITE_URL/#timeline
YOUR_SITE_URL/#analytics
```

## Typography and presentation

The interface uses **DM Sans** with modern system-sans fallbacks. The charts and world-reach visualization use native HTML, CSS, and inline SVG, so no charting library is required.

## Preview locally

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Google Drive access

The original 23 Google Drive file IDs remain in place. Each recording must be shared with the same audience that can view the published page. Video content is hosted on Google Drive and subject to separate access permissions.
